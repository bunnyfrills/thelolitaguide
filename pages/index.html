Hi there! Welcome to my beginner Lolita guide!

This guide was put together to help people starting out in discovering everything they need to know. While r/lolita, facebook groups, and other forums exist, this site functions as a place to reference before asking easily answered questions on those groups. While it is intended for beginner usage, this guide contains a lot of information that might be helpful to more experienced lolitas.
